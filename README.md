# itd_stats
Утилита для быстрого получения статистики по аккаунту с помощью npm

# Быстрый старт
- Пропишите git clone "https://github.com/sneyri/itd_stats" для того чтобы скопировать программу
- После успешного копирования напишите npm install для установки нужных пакетов.

[ВАЖНО! Для того чтобы все работало нужно установить Node.js!]

# Планируется в ближайшее время
Планируется написание подобной утилиты с помощью Python
