# itd_stats
Утилита для быстрого получения статистики по аккаунту с помощью npm

# Быстрый старт
- Пропишите git clone "https://github.com/sneyri/itd_stats" для того чтобы скопировать программу
- После успешного копирования напишите npm install для установки нужных пакетов.
- Создайте файл .cookies и поместите туда refresh_token=ТОКЕН; is_auth=1;
- Пропишите npm start
- После в папке Stats появится статистика

# Где взять refresh_token?
refresh_token можно взять из кук, следуйте по шагам:
- Войдите в итд.com
- Откройте DevTools(F12)
- Перейдите на вкладку Application → Cookies → https://xn--d1ah4a.com
- Найдите куку refresh_token и скопируйте ее значение, его и нужно вставлять в .cookies
Пример содержимого .cookies: refresh_token=eyJhbGciOiJIUzI1NiIs...; is_auth=1; 

*Если по какой то причине нету - обновите страницу.*

**ВАЖНО! Для того чтобы все работало нужно установить Node.js!**
